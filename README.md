# Baixar mods atualizados

1. Entre na pasta do servidor

```sh
cd /Minecraft/Solo11
```

2. Execute o packwiz

```sh
java -jar /Minecraft/packwiz-installer-bootstrap.jar \
http://localhost/packwiz/pack.toml
```
